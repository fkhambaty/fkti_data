### Troubleshooting

**Problem: Website looks broken or unstyled**
- Solution: Make sure `styles.css` is in the same folder as `index.html`

**Problem: Interactive features don't work**
- Solution: Make sure `script.js` is in the same folder as `index.html`
- Check that JavaScript is enabled in your browser

**Problem: Progress not saving**
- Solution: Make sure cookies/local storage is enabled in your browser
- Don't use incognito/private browsing mode

**Problem: Can't see code syntax highlighting**
- Solution: Check your internet connection (CDN resources need internet)
- Alternatively, the website will still work without highlighting

**Problem: Quizzes not loading or showing errors**
- Solution: Make sure `quiz-data.js` is in the same folder as `index.html`
- Refresh the page (F5 or Cmd+R)
- Check browser console for errors (F12)
