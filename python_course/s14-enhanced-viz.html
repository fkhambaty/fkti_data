ğŸ¥§ Chart 1: Pie Chart - Loan Status Distribution

ğŸ“š What Is a Pie Chart?

A pie chart is a circular graph divided into slices, where:

  • The whole circle = 100% of your data
  • Each slice = One category (like "Approved" or "Rejected")
  • Slice size = Percentage of that category

Best for: Showing parts of a whole when you have 2-5 categories.

ğŸ• Pizza Analogy

Imagine sharing a pizza between friends. If Rahul ate 62% and Priya ate 38%, Rahul's slice would be bigger. A pie chart does exactly this with data - bigger slice = bigger percentage!

ğŸ“š What Is a Count Plot?

A count plot is one of the simplest charts. It:

  • Takes a categorical column (like Education)
  • Counts how many rows are in each category
  • Shows result as vertical bars

Formula: X-axis = Categories, Y-axis = Count of each

Ask 100 students: "Are you a graduate?" Draw two bars - one for "Yes" and one for "No". The bar heights show how many said each. That's a count plot!

ğŸ“¦ Chart 4: Box Plot - CIBIL Score by Loan Status

ğŸ“š What Is a Box Plot?

A box plot (box-and-whisker plot) shows the distribution of a numeric variable:

  • Line in middle of box = Median (middle value)
  • Box edges = 25th and 75th percentile (where 50% of data lives)
  • Whiskers = Range of most data
  • Dots outside = Outliers (unusual values)

ğŸ“Š Exam Scores Analogy

Imagine comparing exam scores of Class A vs Class B:

  • Box = Most students scored between these marks
  • Line in box = The "typical" student's score
  • Dots outside = That one genius who scored 100% or someone who scored 10%!

ğŸŒŠ Chart 5: KDE Plot - CIBIL Score Distribution

ğŸ“š What Is a KDE Plot?

KDE stands for Kernel Density Estimate. It's a smooth curve showing where your data is concentrated.

Think of it as a smoothed histogram - instead of jagged bars, you get a beautiful flowing curve!

ğŸ”ï¸ Mountain Analogy

Imagine your data as a landscape:

  • Peaks (high points) = Where most data is concentrated
  • Valleys (low points) = Where data is sparse
  • Width of curve = How spread out your data is

A KDE draws this "data mountain" for you!

# KDE plot of CIBIL scores

    data=df[' cibil_score'],  # The numeric column

plt.title('Distribution of CIBIL Scores', fontsize=14, fontweight='bold')

plt.show()

ğŸ“š What Is a Violin Plot?

A violin plot is the best of both worlds:

  • Shows distribution shape (like KDE)
  • Shows median and quartiles (like box plot)

It's called "violin" because the shape looks like a violin! ğŸ»

ğŸ­ Best of Both Worlds

Box Plot says: "Here's the median and spread"

KDE Plot says: "Here's where data is concentrated"

Violin Plot says: "Why choose? I'll show you BOTH!"

The wider the violin at any point, the more data exists at that value.

âš« Chart 7: Scatter Plot - Income vs Loan Amount

ğŸ“š What Is a Scatter Plot?

A scatter plot shows the relationship between two numbers:

  • Each dot = One loan application
  • X position = First number (Income)
  • Y position = Second number (Loan Amount)
  • Color = Can show a third variable (Approved/Rejected)

ğŸ¯ Pattern Detection

A scatter plot answers: "Do these two things move together?"

  • Dots go up-right â†’ YES! Both increase together (positive correlation)
  • Dots go down-right â†’ Opposite! One goes up, other goes down (negative)
  • Dots are random cloud â†’ No relationship

ğŸ“š What Is a Histogram?

A histogram groups data into bins (ranges) and counts how many fall into each:

  • Data is divided into equal-width ranges (bins)
  • Each bar shows how many values fall in that range
  • Taller bar = More data in that range

ğŸ‚ Age Groups Analogy

Imagine sorting people by age into groups:

  • 0-20 years: 50 people
  • 21-40 years: 120 people â† Tallest bar!
  • 41-60 years: 80 people
  • 61+ years: 30 people

That's a histogram! It shows which ranges are most common.

ğŸ’¡ Histogram Shapes & What They Mean

  • Bell shape â†’ Normal distribution (most common in middle)
  • Skewed right â†’ Most data on left, tail on right (e.g., real income)
  • Flat bars â†’ Uniform distribution (evenly spread)
  • Two humps â†’ Bimodal (two groups in your data)

ğŸ‘€ Key Observation

For this dataset: The bars are roughly equal height â†’ Income is evenly distributed.

Note: Real-world income data usually skews right (most earn moderate, few earn a lot). This suggests the dataset might be synthetic or pre-processed.

ğŸš« Common Mistakes (Enhanced Viz)

  • Too many slices in a pie chart â€” Pie charts get messy with many categories; use a bar chart or group small slices into "Other".
  • Ignoring outliers in box plots â€” Box plots show median and quartiles; points outside the whiskers are outliersâ€”don't remove them without reason.
  • Overlapping elements â€” Use alpha for transparency or jitter so points don't hide each other in scatter/count plots.

import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

df = pd.read_csv('loan_approval_dataset.csv')
sns.set(style='whitegrid')

# 1. PIE CHART - Proportions
plt.pie(df[' loan_status'].value_counts(), labels=['Approved','Rejected'], autopct='%1.1f%%')

# 2. COUNT PLOT - Count categories (v0.14+ syntax)
sns.countplot(x=' education', data=df, hue=' education', legend=False)

# 3. BAR PLOT - Average by category
sns.barplot(x=' education', y=' loan_amount', data=df, hue=' education', legend=False)

# 4. BOX PLOT - Distribution with outliers
sns.boxplot(x=' loan_status', y=' cibil_score', data=df, hue=' loan_status', legend=False)

# 5. KDE PLOT - Smooth distribution
sns.kdeplot(data=df[' cibil_score'], fill=True)

# 6. VIOLIN PLOT - Distribution + box
sns.violinplot(x=' loan_status', y=' cibil_score', data=df, hue=' loan_status', legend=False)

# 7. SCATTER PLOT - Two variables relationship
sns.scatterplot(x=' income_annum', y=' loan_amount', hue=' loan_status', data=df)

# 8. HISTOGRAM - Frequency distribution
plt.hist(df[' income_annum'], bins=20)
# OR with KDE: sns.histplot(data=df, x=' income_annum', kde=True)
