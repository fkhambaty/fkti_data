None represents the absence of a value. It's not zero, not empty string, not False - it's literally "nothing".

    ğŸ“¦ Box with 0 = 0     (zero - a number in the box!)

    or was deliberately set to "nothing"

  • A function that doesn't return anything
  • A variable before you assign a value
  • When data is missing from a database
  • Default value for optional parameters

ğŸš« Common Mistakes (Data Types)

  • Adding string and number without converting â€” "5" + 3 errors; use int("5") + 3 or "5" + str(3).
  • Confusing = and == â€” One equals assigns; two compare. Use if x == 5 not if x = 5.
  • float("3.14") is fine, int("3.14") errors â€” int() needs a whole-number string; convert to float first then int if needed.
