The "Download App" link on the site points to Google Drive—no APK files are stored in this folder.

To update the app for users:
1. Build the APK in Android Studio (Build → Build Bundle(s) / APK(s) → Build APK(s)).
2. Upload the APK to the shared Google Drive folder (link used in the nav: DataForDummies Android App folder). There should be one APK in that folder for users to download.

For more options (e.g. GitHub Releases), see APK_DOWNLOAD_SETUP.md in the Website folder.
