What is a Transformer?

2017 architecture replacing RNNs with self-attention. Enables parallel processing. Foundation of GPT, BERT, all modern LLMs.

2

Attention mechanism?

Computes relevance between token pairs. Query-Key-Value: score = softmax(QK^T/sqrt(d)) * V. Multi-head for parallel attention.

3

Fine-tuning vs RAG?

Fine-tuning: retrain model weights on your data. RAG: feed relevant docs at query time. RAG is faster, no training needed.

4

Prompt engineering?

Crafting inputs for better outputs. Techniques: few-shot, chain-of-thought, system prompts, output format specification.

5

Evaluating LLM outputs?

BLEU/ROUGE for similarity, perplexity for quality, human eval for subjective, LLM-as-judge for scalable evaluation.
