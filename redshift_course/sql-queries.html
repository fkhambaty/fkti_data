📊 Redshift SQL

Redshift SQL is PostgreSQL-compatible with some powerful additions:

-- Window functions for analytics
SELECT
    date,
    revenue,
    SUM(revenue) OVER (ORDER BY date ROWS UNBOUNDED PRECEDING) AS running_total,
    LAG(revenue, 7) OVER (ORDER BY date) AS last_week,
    PERCENT_RANK() OVER (ORDER BY revenue) AS percentile
FROM daily_sales;

-- Approximate count distinct (much faster!)
SELECT APPROXIMATE COUNT(DISTINCT user_id) FROM events;

-- SUPER type for semi-structured data (JSON)
SELECT json_data.customer.name
FROM orders
WHERE json_data.customer.country = 'US';
