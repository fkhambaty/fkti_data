🌊 Redshift Spectrum

Query data directly in S3 without loading it into Redshift. Perfect for cold/historical data.

-- Create external schema pointing to S3
CREATE EXTERNAL SCHEMA spectrum_schema
FROM DATA CATALOG
DATABASE 'my_glue_db'
IAM_ROLE 'arn:aws:iam::123456:role/SpectrumRole';

-- Query S3 data like a regular table!
SELECT date, count(*) AS events
FROM spectrum_schema.raw_events
WHERE date >= '2024-01-01'
GROUP BY date;

🧒 ELI5: The Library Card

Instead of buying every book (loading all data), you get a library card (Spectrum) that lets you read any book in the library (S3) without taking it home. You only pay for the pages you actually read!
